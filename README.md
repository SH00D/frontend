# 💊 Интернет-аптека (Курсовой проект)

## 📌 О проекте

Веб-приложение интернет-аптеки, разработанное в рамках курсовой работы.
Проект демонстрирует реализацию пользовательского интерфейса для просмотра товаров и оформления заказа.

⚠️ Проект носит исключительно учебный характер и не предназначен для реального использования.

---

## 🚀 Функционал

* Каталог товаров
* Карточка товара
* Поиск и фильтрация
* Корзина (добавление, удаление, изменение количества)
* Форма обратной связи

---

## 🛠️ Технологии

### Frontend:

* HTML5
* CSS3
* JavaScript (Vanilla JS)

### Backend:

* Java
* Spring Boot

---

## 📁 Структура проекта

```
css/        — стили  
js/         — логика  
img/        — изображения  
*.html      — страницы сайта  
backend/    — серверная часть (Spring Boot)  
```

---

## ⚙️ Запуск проекта

### Frontend:

Открыть файл `index.html` в браузере
или запустить через локальный сервер

### Backend:

```
cd backend
./mvnw spring-boot:run
```
